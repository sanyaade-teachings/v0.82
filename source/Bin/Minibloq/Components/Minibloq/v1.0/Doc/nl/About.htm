Dankwoord

Dit project is mede tot stand gekomen door de hulp en bijdragen van een groot aantal personen. Mocht ik iemand vergeten zijn te noemen, bij deze alvast mijn excuses! Het is mogelijk om een bericht achter te laten op mijn blog http://minibloq.org (in elk geval zolang het project nog actief is).
RobotGroup
 RobotGroup is een Argentijns bedrijf in robotica. Hier maken we robots, controllers en een hoop andere dingen. Daarnaast hebben we in ons RG.Labs open source projecten zoals DuinOS ontwikkeld en uitgebracht.


FONSOFT
 Dit project is onderdeel van een veel groter project, genaamd XDF, waarvoor we subsidies ontvangen hebben van FONSOFT, wat onderdeel is van Agencia Nacional de Promoción Científica y Tecnológica, wat valt onder de Argentijse overheid. XDF is FONSOFT's project NA 134/07.

Open source software
 Minibloq is volledig gebaseerd op open source software. Het is samengesteld uit een groot aantal sub-pakketten, welke geheel of gedeeltelijk gebruikt zijn in Minibloq. Al deze pakketten vallen onder hun eigen licentie, welke ondermeer terug te vinden is in de licentie directory en alle sub-directories eronder. De volgende pakketten zijn geheel of gedeeltelijk gebruikt voor de ontwikkeling van Minibloq:
