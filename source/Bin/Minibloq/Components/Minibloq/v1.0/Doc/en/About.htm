Attributions and Credits

This project is the result of a lot of work and contributions. If I'm forgetting someone, sorry! (and please remember that you can go to the blog http://minibloq.org and leave a message -at least while the project is alive-):
RobotGroup
 RobotGroup is an Argentine robotics company. There we develop robots, controllers and many more things. Another fun place at RobotGroup is RG.Labs, where we developed and released open source projects like DuinOS.


FONSOFT
 This project is part of a bigger project, called XDF, which received funds from the FONSOFT, which belongs to the Argentine Government's Agencia Nacional de Promoción Científica y Tecnológica. The XDF is the FONSOFT's project NA 134/07.

Open source software
 Minibloq is completely based on open source software. It is also component-based software. Therefore it's composed by a lot of sub-packages. All those packages may include their own license files.
