Atribusi dan Kredit

Proyek ini adalah hasil dari sejumlah kerja dan kontribusi. Jika saya melupakan beberapa orang, maaf! (dan silahkan mengingatkannya yaitu dengan mengunjungi blog http://minibloq.org dan tinggalkan sebuah pesan -di bagian akhir selagi proyek masih berjalan-):
RobotGroup
 RobotGroup adalah perusahaan robotika Argentina. Disana kami mengembangkan robot, controller serta banyak hal lainnya. Tempat menyenangkan lainnya di RobotGroup adalah RG.Labs, dimana kami kembangkan dan merilis proyek-proyek sumber terbuka (open source) seperti DuinOS.


FONSOFT
 Proyek ini adalah bagian dari proyek yang lebih besar, disebut XDF, yang menerima dana dari FONSOFT, termasuk Pemerintah Argentina Agencia Nacional de Promoción Científica y Tecnológica. XDF merupakan proyek NA 134/07 FONSOFT.

Open source software
 Minibloq secara lengkap merupakan perangkat lunak berbasis sumber terbuka. Ini juga merupakan perangkat lunak berbasis komponen. Oleh karena itu disusun dengan banyak sub-paket. Seluruh paket tersebut termasuk file lisensinya masing-masing. Beberapa komponn mengandung kode, pustaka atau, dalam beberapa hal, paket software keseluruhan dari proyek-proyek berikut (silahkan lihat folder lisensi and the subfolders belonging to each component in the current Minibloq distribution):
