Atribuiçõs e Créditos

Este projeto é o resultado de muito trabalho e contribuições. Se estou esquecendo alguém, perdoem-me" (e por favor lembre-se que você pode ir até o blog http://minibloq.org e deixar uma mensagem -pelo menos enquanto o projeto estiver vivo-):
RobotGroup
 A RobotGroup é uma companhia argentina de robótica. Desenvolvemos robôs, controladores e muitas outras coisas. Outro lugar divertido na RobotGroup é RG.Labs, onde desenvolvemos e lançamos projetos de código aberto como o DuinOS.


FONSOFT
 Este projeto é parte de outro grande projeto, chamado XDF, o qual recebeu fundos da FONSOFT, pertencente a Agencia Nacional de Promoción Científica y Tecnológica do Governo da Argentina. O XDF é o projeto NA 134/07 da FONSOFT.

Software livre
 Minibloq é baseado completamente em software de código aberto. Ele também é construído com software baseado em componentes. Assim ele é composto por vários pacotes secundários. Todos esses pacotes podem incluir suas próprias licenças. Alguns componentes podem incluir linhas de código, bibliotecas ou, em alguns casos, todo o pacote do software originado dos seguintes projetos (por favor, veja a pasta de licenças e as subpastas correspondentes a cada componente na distribuição atual do Minibloq):
