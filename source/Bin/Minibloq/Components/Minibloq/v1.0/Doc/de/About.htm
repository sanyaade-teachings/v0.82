RobotGroup
 RobotGroup ist eine angentinische Robotik Firma. Dort entwicklen wir Roboter, Steuerungen und viele andere Sachen. Eine andere interessante Abteilung von RobotGroup ist RG.Labs, wo wir Open Source Projekte wie DuinOS entwickeln und veröffentlichen.


FONSOFT
 Dieses Projekt ist Teil eines größeren Projektes, genannt XDF, welches von der FONSOFT finanziert wurde, die der argentinischen RegierungsagenturAgencia Nacional de Promoción Científica y Tecnológica gehört. XDF ist FONSOFT's Project Nr. 134/07.

Open Source Software
 Minibloq basiert vollständig auf quelloffener Software. Es ist außerdem komponentenbasierte Software. Deshalb besteht es aus eine Menge von Unterpaketen. Alle diese Pakete können eigene Lizenzdateien enthalten. Einige von den Komponenten können Code, Bibliotheken oder, in einigen Fällen, komplette Softwarepakete der folgenden Projekte enthalten (siehe Lizenzverzeichnis und Unterverzeichnisse der jeweiligen Komponenten, die in der aktuellen Minibloq Distribution enthalten sind):
