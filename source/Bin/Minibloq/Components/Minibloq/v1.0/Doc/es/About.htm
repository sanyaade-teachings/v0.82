Atribuciones y créditos

Este proyecto es el resultado de mucho trabajo y de diferentes contribuciones. Si estoy olvidando a alguien, pido disculpas (y por favor recuerda que puedes dirigirte a la página web http://minibloq.org y dejar un mensaje -al menos mientras el proyecto siga activo-):
RobotGroup
 RobotGroup es una empresa argentina de robótica. Allí desarrollamos robots, controladores y muchas otras cosas. Otro lugar divertido en RobotGroup es RG.Labs, donde desarrollamos y publicamos proyectos de código abierto (open source) como  DuinOS.


FONSOFT
 Este proyecto es parte de un proyecto aún mayor, llamado XDF, el cual recibió fondos del FONSOFT, el cual pertenece a la institución argentina gubernamental Agencia Nacional de Promoción Científica y Tecnológica. El XDF es el proyecto NA 134/07 (FONSOFT).

Software de código abierto (open source)
 Minibloq está basado completamente en software de código abierto. Es además software basado en componentes. Por lo tanto, está compuesto por diferentes paquetes. Cada uno de estos paquetes puede incluir su propio archivo de licencia. Algunos de los componentes pueden ser código, librerías o, en algunos casos, el paquete completo, de los siguientes proyectos (por favor mira la carpeta de licencias y las subcarpetas pertenecientes a cada componente en la distribución actual de Minibloq):
