Atribuciones y créditos

Este proxecto é ol resultado de moito traballo e de diferentes contribucións. De estar a esquecer a alguén, pido desculpas (e por favor, lembra que podes dirixirte á páxina web http://minibloq.org e deixar unha mensaxe –cando menos, mentres siga activo o proxecto-):
RobotGroup
 RobotGroup é unha empresa arxentina de robótica. Alí desenvolvemos robots, controladores e moitas outras cousas. Outro lugar divertido en RobotGroup é RG.Labs, onde desenvolvemos e publicamos proxectos de código aberto (open source) como DuinOS.


FONSOFT
 Este proxecto é parte dun proxecto aínda maior, chamado XDF, o cal recibiu fondos do FONSOFT, á súa vez pertencente á institución arxentina gobernamental Agencia Nacional de Promoción Científica y Tecnológica. El XDF é o proxecto NA 134/07 (FONSOFT).

Software de código aberto (open source)
 Minibloq está baseado completamente en software de código aberto. É ademais software baseado en compoñentes. Polo tanto, está composto por diferentes paquetes. Cada un destes módulos pode incluír o seu propio ficheiro de licenza. Algúns dos compoñentes poden ser código, librarías ou, nalgúns casos, o paquete completo, dos seguintes proxectos (por favor olla o cartafol de licenzas e os cartafoles pertencentes a cada compoñente na distribución actual de Minibloq):
